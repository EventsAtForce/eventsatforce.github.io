Requirement of this repo: Used to redirect the incoming bitly link. An illustration of how it works-

bit.ly/EventsAtFORCE
          or            ---------->   eventsatforce.github.io    --------------> To any site that FORCE operates with.
bit.ly/eventsatforce                                             [Auto-redirect]


The sole application of this repo is to redirect the incoming traffic to whatever the current FORCE website is.
The design of this site would be a bit more functional if we have a spinners in the center of the site. We can make use
of the Bootstrap Spinners. Please note that for ancient browsers there will be an issue because they don't handle
redirect properly. But it's 2020 and I don't think there are any devices which use that old browsers.
